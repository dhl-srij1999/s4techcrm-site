Documentation

Quickstart

  1. Install Docker and Docker Compose.
  2. Run docker-compose -f docker-compose.test.yml build.
  3. Run ./run-e2e-tests.sh to start Postgres, run migrations, and tests.

Manual Install

  1. Install Node.js and Postgres.
  2. Set DATABASE_URL and JWT_SECRET.
  3. Run migrations and start the app.
